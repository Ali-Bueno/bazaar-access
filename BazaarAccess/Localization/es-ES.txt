# BazaarAccess - Español de España (es-ES)
#
# Traducción del catálogo de referencia en-US.txt.
# El mod recurre al inglés para cualquier clave que falte en una traducción, así que una
# traducción parcial es válida: traduce lo que puedas y deja el resto fuera.
#
# Formato
#   clave = texto
#   Las líneas que empiezan por '#' son comentarios. Las líneas en blanco se ignoran.
#   {0}, {1}... son marcadores que rellena el mod (el nombre de un objeto, una cantidad...).
#   Conserva todos los marcadores que aparezcan en el texto inglés; puedes reordenarlos
#   si tu idioma necesita otro orden de palabras.
#   Usa \n para un salto de línea.
#
# Cantidades
#   Las claves acabadas en .one / .other se eligen según la cantidad: .one para una sola
#   cosa, .other para cualquier otra cantidad. Los idiomas que no necesiten la distinción
#   pueden dar el mismo texto a las dos.
#
# Palabras como Escudo, Quemadura, Bronce o Arma NO están aquí: el mod las toma del propio
# juego, ya traducidas, para decir exactamente la misma palabra que la carta.
