# BazaarAccess - Italiano (it-IT)
#
# Questo file è la traduzione italiana del catalogo di riferimento (en-US.txt).
# La mod ricade sull'inglese per ogni chiave mancante nella traduzione, quindi
# una traduzione parziale va bene: traduci ciò che puoi e lascia perdere il resto.
#
# Formato
#   chiave = testo
#   Le righe che iniziano con '#' sono commenti. Le righe vuote sono ignorate.
#   {0}, {1}... sono segnaposto che la mod riempie (un nome oggetto, una quantità...).
#   Mantieni ogni segnaposto presente nel testo inglese - puoi riordinarli
#   se la tua lingua richiede un ordine delle parole diverso.
#   Usa \n per andare a capo.
#
# Conteggi
#   Le chiavi che finiscono in .one / .other sono scelte in base alla quantità: .one per
#   una singola cosa, .other per qualsiasi altra quantità. Le lingue che non hanno bisogno
#   della distinzione possono dare lo stesso testo a entrambe.
#
# Parole come Shield, Burn, Bronze o Weapon NON sono qui: la mod le prende direttamente
# dal gioco, già tradotte, così dice esattamente la stessa parola della carta.
