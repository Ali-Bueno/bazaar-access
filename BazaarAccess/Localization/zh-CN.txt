# BazaarAccess - 简体中文 (zh-CN)
#
# 本文件是参考目录（en-US.txt）的翻译版本。
# 如果缺少某个键的翻译，模组会回退到英文，所以部分翻译也没问题——能翻多少翻多少，其余留空即可。
#
# 格式
#   key = 文本
#   以“#”开头的行是注释。空行会被忽略。
#   {0}、{1}……是模组填入的占位符（物品名称、数量等）。
#   保留英文文本中出现的所有占位符——如果语言的语序需要，可以调整它们的顺序。
#   换行请用 \n。
#
# 数量
#   以 .one / .other 结尾的键根据数量选择：单数用 .one，其余情况用 .other。
#   不需要区分单复数的语言可以让两者使用相同的文本。
#
# 像 Shield、Burn、Bronze、Weapon 这样的词不在本文件中：模组直接从游戏本身获取这些已翻译好的词，
# 因此念出的词与卡牌上显示的完全一致。
