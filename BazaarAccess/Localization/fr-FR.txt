# BazaarAccess - Français (fr-FR)
#
# Ceci est une traduction du catalogue de référence anglais (en-US).
# Le mod utilise l'anglais par défaut pour toute clé manquante dans une traduction,
# une traduction partielle convient donc - traduisez ce que vous pouvez et laissez le reste de côté.
#
# Format
#   clé = texte
#   Les lignes commençant par '#' sont des commentaires. Les lignes vides sont ignorées.
#   {0}, {1}... sont des paramètres que le mod remplit (un nom d'objet, un montant...).
#   Conservez tous les paramètres présents dans le texte anglais - vous pouvez les réordonner
#   si votre langue nécessite un ordre des mots différent.
#   Utilisez \n pour un retour à la ligne.
#
# Comptages
#   Les clés se terminant par .one / .other sont choisies selon le montant : .one pour un seul élément,
#   .other pour tout autre montant. Les langues qui n'ont pas besoin de cette distinction peuvent donner
#   le même texte aux deux.
#
# Les mots comme Bouclier, Brûlure, Bronze ou Arme ne sont PAS ici : le mod les récupère
# directement depuis le jeu, déjà traduits, pour dire exactement le même mot que la carte.
