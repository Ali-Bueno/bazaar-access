# BazaarAccess - Português (pt-BR)
#
# Este é o catálogo de referência traduzido a partir do en-US.txt.
# O mod usa o inglês como alternativa para qualquer chave que falte na tradução,
# então uma tradução parcial funciona - traduza o que puder e deixe o resto de fora.
#
# Formato
#   key = texto
#   Linhas que começam com '#' são comentários. Linhas em branco são ignoradas.
#   {0}, {1}... são espaços reservados preenchidos pelo mod (nome de um item, uma quantidade...).
#   Mantenha todos os espaços reservados que aparecem no texto em inglês - você pode reordená-los
#   se o seu idioma precisar de uma ordem de palavras diferente.
#   Use \n para quebra de linha.
#
# Contagens
#   Chaves terminadas em .one / .other são escolhidas pela quantidade: .one para uma coisa só,
#   .other para qualquer outra quantidade. Em português, zero também usa a forma singular -
#   o mod já trata isso, então basta traduzir .one como singular.
#
# Palavras como Shield, Burn, Bronze ou Weapon NÃO estão aqui: o mod pega essas palavras
# do próprio jogo, já traduzidas, então ele fala exatamente a mesma palavra que a carta usa.
