# BazaarAccess - Deutsch (de-DE)
#
# This is the reference catalogue: every other language is a translation of this file.
# The mod falls back to English for any key a translation is missing, so a partial
# translation is fine - translate what you can and leave the rest out.
#
# Format
#   key = text
#   Lines starting with '#' are comments. Blank lines are ignored.
#   {0}, {1}... are placeholders the mod fills in (an item name, an amount...).
#   Keep every placeholder that appears in the English text - you may reorder them
#   if your language needs a different word order.
#   Use \n for a line break.
#
# Counts
#   Keys ending in .one / .other are picked by the amount: .one for a single thing,
#   .other for any other amount. Languages that don't need the distinction can give
#   both the same text.
#
# Words like Shield, Burn, Bronze or Weapon are NOT here: the mod takes those from the
# game itself, already translated, so it says exactly the same word the card does.
